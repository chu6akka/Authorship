# Authorship MVP (легкая версия)

Минимальный аналитический MVP для сравнения:
- полностью идентичных формулировок;
- n-грамм;
- синонимизмов (базово);
- перефраза (кандидаты);
- транслитерации;
- маркеров интернет-коммуникации.

## Быстрый старт

```bash
python authorship_mvp.py --query tests/data/query.txt --sample tests/data/sample.txt --report report.md
```

После запуска будет создан `report.md` с черновиком аналитического отчета.

> Важно: результат — вспомогательная аналитика, а не автоматический процессуальный вывод.
